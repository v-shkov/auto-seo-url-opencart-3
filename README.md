# auto-seo-url
Автоматическая генерация SEO URL товаров и категорий для Open Cart 3

Расширение позволяет значительно облегчить работу администратора интернет-магазина, позволяя автоматически создавать SEO URL товаров и категорий путём транслитерации наименования товара или категории.

Почему это важно?

Если не делать SEO URL, то ссылка на товар или категорию будет иметь вид "product_id=3717", что является не читаемым для поисковых роботов, что в свою очередь способствует ухудшению позиций вашего сайта в поисковой выдаче.

Что умеет модуль:

При ручном создании карточки товара, категории в момент сохранения внесенный данных, если поле SEO URL не заполнено, то автоматически делает транслитерацию наименования товара и подставляя значения в поле SEO URL.
Если в вашем интернет-магазине уже есть товар или категорий с аналогичным URL, то на странице появится ошибка, которая уведомит вас о том, что данный URL уже используется.

* Отображение ошибки при использовании одинакового SEO URL

В данном случае, вы можете вручную прописать нужный вам SEO URL, в случае повторной ошибки, модуль уведомит вас снова об ошибке.

ОСНОВНОЕ ПРЕИМУЩЕСТВО:

При работе с большим количеством товаров, когда нужно постоянно добавлять большое количество товаров (сотни, тысячи товаров за раз) путём выгрузки через EXCEL, CSV, либо используя сторонние программы, которые добавляют товары путём запросов к базе данных, минуя основную логику работы сайта. Возникает необходимость для всех товаров создать SEO URL, вот тут и начинается поиск решений, чтобы сделать это с минимальными усилиями. Ведь согласитесь, не всегда хочется тратить время на это, проще автоматизировать и заниматься более важными делами.

Для этих целей был и разработан модуль, теперь, в административной части вашего интернет-магазина, появляется отдельная кнопка "Генерация SEO URL", которая открывает страницу с одной заветной кнопкой, которая способна в разы облегчить Ваш труд.

Модуль позволяет генерировать SEO URL ручным способом, путём нажатия на кнопку, либо же, предоставляет возможность более опытным пользователям настраивать автоматическую генерацию SEO URL с помощью CRON (по определенному вами времени будет запускать скрипт, который анализирует незаполненные SEO URL у товаров и категорий и если такие есть, автоматически создаёт их).

Бывают ситуации, когда два или более товаров имеют одинаковое наименование, в таком случае модуль автоматически подставит дополнительный идентификатор, разделенный символом "-", можно не беспокоиться о том, что несколько одинаковых товаров будут иметь одинаковые URL.

Установка для OpenCart 3.x:

В административной панели OpenCart выполните следующие действия:

Расширения -> Установить расширение
Обновить модификаторы
Перейдите в Система -> Пользователи -> Группа пользователей -> Администратор
Разрешить для модуля просмотр и внесение изменений
После этих шагов модуль "Генерация SEO URL" должен появиться в меню "Система" -> "Обслуживание" -> "Генерация SEO URL"
